# S.15 Make a Feature-Full Data Table

## Building the Rows

```jsx
function Table({ data }) {
	const renderedRows = data.map((fruit) => {
		return (
			<tr key={fruit.name}>
				<td>{fruit.name}</td>
				<td>{fruit.color}</td>
				<td>{fruit.score}</td>
			</tr>
		);
	});

	return (
		<table>
			<thead>
				<tr>
					<th>Fruit</th>
					<th>Color</th>
					<th>Score</th>
				</tr>
			</thead>
			<tbody>{renderedRows}</tbody>
		</table>
	);
}

export default Table;
```
